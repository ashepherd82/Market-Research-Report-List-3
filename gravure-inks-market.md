<p><p>Gravure inks are classified into different types based on their composition and solvent used. Water type gravure inks are water-based, environmentally friendly inks. Alcoholic type gravure inks contain alcohol as a solvent, while benzene type gravure inks use benzene as a solvent. Petrol type gravure inks are solvent-based inks that use petroleum distillates as a solvent. Each type of gravure ink has its own set of advantages and is used in various printing applications based on their properties and requirements.</p></p>
